## DFS(Depth-First Search, 깊이 우선 탐색)

- BFS보다 좀 더 간단하지만, 속도는 좀 더 느리다.
- 일반적으로 BFS에 비해 널리 쓰인다.
- 자기 자신을 호출하는 재귀의 형태를 가지고 있다.

## BFS(Breadth-First Search, 너비 우선 탐색)

- 방문한 노드를 순서대로 저장할 수 있는 큐를 주로 사용한다.
- 다익스트라 알고리즘과 유사하다.
